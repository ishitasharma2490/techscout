# 💻 TechScout — Remote Tech Job Finder

Welcome to **TechScout** — a web application built to make remote tech job searching easier, cleaner, and more organized. 🚀

Instead of going through multiple websites and scattered listings, TechScout brings remote tech opportunities into one simple interface where users can search and explore jobs more efficiently.

This project is developed as part of a web development assignment to demonstrate the use of **JavaScript**, **API integration**, **array higher-order functions**, and **responsive UI design**.

---

## 🎯 Purpose of the Project

The purpose of **TechScout** is to help users discover remote tech jobs in a user-friendly way. The app fetches real-time job data from a public API and displays it dynamically on the webpage.

This project also demonstrates important web development concepts such as:
- fetching data from an API
- displaying dynamic content on a webpage
- implementing search and sorting
- building a responsive and visually appealing interface

---

## 🌐 API Used

This project uses the **Arbeitnow Job Board API** to fetch job listings.

The API provides job-related information such as:
- job title
- company name
- location
- job description
- application link

---

## ✨ Features

- 🔍 **Search jobs** by title
- ↕️ **Sort jobs** (A–Z / Z–A)
- 🌙 **Dark mode toggle**
- 📡 **Fetch and display real-time API data**
- ⏳ **Loading indicator** while fetching data
- ⚠️ **Error handling** for failed API requests
- 📱 **Responsive design**
