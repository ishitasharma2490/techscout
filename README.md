# 💻 TechScout — Remote Tech Job Finder

Welcome to **TechScout** — a web application built to make remote tech job searching easier, cleaner and more organized. 🚀

Instead of going through multiple websites and scattered listings, TechScout brings remote tech opportunities into one simple interface where users can search, filter, and explore jobs more efficiently.

This project is being developed as part of a web development assignment to demonstrate the use of **JavaScript**, **API integration**, **array higher-order functions**, and **responsive UI design**.

---

## 🎯 Purpose of the Project

The purpose of **TechScout** is to help users discover remote tech jobs in a user-friendly way. The app will fetch real-time job data from a public API and display it dynamically on the webpage.

This project will also demonstrate important web development concepts such as:
- fetching data from an API
- displaying dynamic content on a webpage
- implementing search, filtering, and sorting
- building a responsive and visually appealing interface

---

## 🌐 API Used

This project uses the **Himalayas Jobs API** to fetch remote job listings.

The API provides job-related information such as:
- job title
- company name
- category
- location / remote status
- description
- application link

---

## ✨ Planned Features

TechScout will include the following features:

- 🔍 **Search jobs** by keyword
- 🧩 **Filter jobs** based on relevant categories
- ↕️ **Sort job listings** by title, company, or other criteria
- 💼 **Display fetched jobs dynamically** in a clean card layout
- ⏳ **Loading state** while data is being fetched
- ⚠️ **Basic error handling** for failed API requests
- 📱 **Responsive design** for mobile, tablet, and desktop screens
- ❤️ Optional feature: **save favorite jobs**
- 🌙 Optional feature: **dark mode / light mode toggle**
